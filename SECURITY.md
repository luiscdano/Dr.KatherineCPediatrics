# Security Policy

## Supported Versions

Se da soporte de seguridad a la rama principal (`main`) y a la version publicada mas reciente.

## Reporting a Vulnerability

Si detectas una vulnerabilidad, reportala de forma privada a:

- `contacto@drkatherinecpediatrics.com`

Incluye:

1. Descripcion del problema y riesgo.
2. Pasos para reproducir.
3. Evidencia tecnica (capturas, logs o payloads).
4. Recomendacion de mitigacion si la tienes.

No publiques vulnerabilidades en issues publicos del repositorio.

## Response Targets

1. Confirmacion de recibido: dentro de 72 horas.
2. Evaluacion inicial: dentro de 7 dias.
3. Plan de correccion: segun severidad y alcance.
